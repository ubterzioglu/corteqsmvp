## Güvenlik modeli

- Tablolar RLS'li; yalnız `is_admin(auth.uid())` SELECT policy'si. Tarayıcıdan INSERT/UPDATE yok.
- `admin_*` RPC'ler SECURITY DEFINER + `service_finder_require_admin()`; `worker_*` RPC'ler
  yalnız `service_role`'a granted (worker `SUPABASE_SERVICE_ROLE_KEY` ile bağlanır).
- **Sırlar DB'de/UI'da tutulmaz**: `provider_configs.secret_ref` yalnızca worker ortamındaki
  env değişkeninin ADIDIR (`TAVILY_API_KEY` vb.).
- robots.txt zorunlu: worker, ekstraksiyondan önce host başına robots kurallarını değerlendirir;
  engellenen kaynak `blocked_robots` olarak işaretlenir, sağlayıcıya hiç gönderilmez.

## Bütçe modeli

| Korkuluk | Yer | Davranış |
|---|---|---|
| Aylık sağlayıcı tavanı | `provider_configs.monthly_cap_usd` | bilgilendirici (v1'de otomatik kapatma yok) |
| İş soft cap | `jobs.soft_cap_usd` | basic depth + SerpAPI fallback kapalı + Flash-Lite |
| İş hard cap | `jobs.hard_cap_usd` | `worker_record_service_finder_cost` dönüşünde anında `budget_stopped` |

Fiyat sabitleri: `workers/service-finder/src/costs.ts` (Tavily $0.008/kredi; SerpAPI $0.015/arama;
Gemini Flash-Lite $0.10/$0.40 per 1M token). Fiyat değişiminde yalnız bu dosya güncellenir.

## Rollback

- Sağlayıcıları kapat: `update service_finder_provider_configs set is_enabled=false;`
- Coolify App 2'yi durdur (web uygulaması etkilenmez).
- UI'ı gizlemek gerekirse: nav grubunu `admin-navigation-registry.ts`'ten çıkar (route'lar kalabilir).
- Yayın sorunu varsa yalnız `admin_publish_service_finder_candidate`'ı devre dışı bırak
  (`revoke execute ... from authenticated`).
- Tabloları HEMEN düşürme — iş/maliyet/olay kayıtları tanı için saklanır.
